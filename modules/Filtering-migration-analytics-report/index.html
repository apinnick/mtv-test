You can filter the results of a Migration Analytics report to display virtual machines according to provider, cluster, migration effort, and other parameters.

Procedure

  1. Click VM migration inventory in the Migration Analytics report.

  2. Click Filter and select a filter type, for example, Provider, OS type, or Rules.

  3. Click Filter by <filter_type> and select one or more options.

  4. If you are filtering by VM name, enter the text in the search field.

    The filtered virtual machines are displayed.
