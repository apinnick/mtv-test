You can install the {project-first} by using the {virt} web console.

Prerequisites

  • Admin privileges

Procedure

  1. In the {virt} web console, click Operators → OperatorHub.

  2. Use the Filter by keyword field to find the {project-short} Operator.

  3. Select the {project-short} Operator and click Install.

  4. On the Install Operator page, click Install.

    On the Installed Operators page, the {project-short} Operator appears in the openshift-migration project with the status Succeeded.

  5. Click {project-short} Operator.

  6. Under Provided APIs, locate the VirtController tile, and click Create Instance.

  7. Click Create.

  8. Click Workloads → Pods to verify that the Provider Inventory, Validation, User Interface, and Controller pods are running.
